# 3x3 Rubik's Cube Simulator

一个使用 C++ 和 ncurses 库编写的 3D 魔方模拟器，可在终端中运行。

## 功能特性

- 🎮 完整的3D魔方模拟
- 🖥️ 终端图形界面（使用ncurses）
- 🔄 实时3D旋转和视角控制
- 🎨 彩色显示（支持256色终端）
- ⚡ 平滑动画效果
- 🎲 随机打乱功能
- 🔧 重置功能

## 系统要求

- Linux系统（推荐Arch Linux或Ubuntu）
- C++17兼容的编译器（g++或clang++）
- ncurses库
- CMake 3.10+
